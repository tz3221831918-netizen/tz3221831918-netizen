# Building SwipeX for macOS

I am building SwipeX, a trackpad-first macOS window manager designed to make window control feel closer to the window itself.

SwipeX focuses on title-bar swipe gestures, Smart Fill, Window Layouts, Quick Access, and a quieter way to manage repeated Mac workspaces.

## What SwipeX Is About

SwipeX is built around a simple idea: window management is spatial, so the interaction should feel spatial too.

Instead of memorizing a large shortcut map or dragging windows to screen edges, SwipeX lets you use title-bar swipe gestures to snap, maximize, reveal, and restore windows on macOS.

## Current Focus

- Trackpad-first window management for macOS
- Saved Window Layouts for repeated work setups
- Quick Access for apps, websites, folders, and files
- Better workflows for MacBook and multi-display users
